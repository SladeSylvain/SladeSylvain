👋 Sobre mí
Soy un Python Developer y Cloud Engineer con base en Santiago de Chile, especializado en construir backends robustos con Django y desplegar infraestructura en AWS. Combino una mentalidad analítica con atención al detalle para desarrollar soluciones que no solo funcionan, sino que escalan.
Actualmente me encuentro en proceso de certificación AWS Cloud Practitioner y sigo profundizando en prácticas de DevOps y arquitectura cloud-native.
Lo que me diferencia:

🇫🇷 Francés C1 — trabajo efectivo con equipos y clientes francófonos.
🌐 Inglés técnico B2 — documentación, issues y PRs sin problema.
☁️ AWS re/Start completado — infraestructura real, no solo teoría.


🚀 Proyectos Destacados
🛒 Portfolio Web con Carrito de Compras

Aplicación web full-stack que combina un portafolio profesional con funcionalidad de e-commerce.

Problema que resuelve: Demostrar capacidades full-stack en un solo proyecto cohesivo — autenticación, manejo de sesiones, y lógica de negocio real.
Stack: Python · Django · PostgreSQL · HTML5/CSS3 · JavaScript · AWS EC2
Highlights:

Autenticación completa con sistema de roles y permisos (ModelForms, admin panel customizado)
Carrito de compras con gestión de estado en sesión
Desplegado en AWS con HTTPS via ACM + ALB
